# Learn-CFA-with-Python
在学习金融与经济知识的过程中，我发现课本上的理论、公式、模型虽然极具价值，但往往抽象、晦涩、难以直观理解。为了真正吃透这些知识，而不是死记硬背，我开始尝试：把课本上的每一个核心知识点，都用 Python 写成可运行的代码。 通过代码实现、数据验证、图表可视化，我把枯燥的理论变成了看得见、算得清、能复现的工具，让学习从 “被动记忆” 变成 “主动实践”，大幅提升理解深度和学习效率。
In learning finance and economics, I found that while the theories, formulas, and models in textbooks were extremely valuable, they were often abstract, obscure, and difficult to understand intuitively. To truly grasp this knowledge instead of rote memorization, I started by writing runnable Python code for every core concept in the textbook. Through code implementation, data verification, and chart visualization, I transformed dry theory into a visible, calculable, and reproducible tool, shifting learning from "passive memorization" to "active practice," significantly improving the depth of understanding and learning efficiency.

本仓库适合：
金融 / 经济专业学生
CFA/FRM/ 考研备考者
想通过代码理解理论的自学者
希望打通 “理论→代码→应用” 全链路的学习者

This repository is suitable for:
Finance/Economics students
CFA/FRM/Graduate school entrance exam candidates
Self-learners who want to understand theory through code
Learners who wish to bridge the entire learning gap from "theory → code → application"

核心目标
化抽象为具体：把课本公式、模型直接写成 Python 代码，所见即所得
强化理解：通过运行、修改代码，真正吃透理论背后的逻辑
可视化学习：用图表展示经济 / 金融模型变化，直观易懂
实用落地：代码可直接用于作业、课题、数据分析、小项目
持续积累：构建一套属于自己的金融经济学代码库

Core Objectives:

Transforming Abstraction into Concreteness: Directly translate textbook formulas and models into Python code for a WYSIWYG experience.
Enhancing Understanding: Truly grasp the logic behind the theories by running and modifying the code.
Visualizing Learning: Use charts to illustrate changes in economic/financial models for intuitive and easy understanding.
Practical Application: The code can be directly used for assignments, research projects, data analysis, and small projects.
Continuous Accumulation: Build your own financial economics codebase.
